# clipFile - Copy your files content to clipboard from terminal

## How to install
### From source:
```bash
# install xclip if on linux
sudo apt install xclip

# clone the repo
git clone https://github.com/vccolombo/clipFile.git
cd clipFile

# install clipFile
python3 setup.py install
```
